# Mob timer

## Getting started

1. `npm install`
1. `npm run update`

## Development

Start the application in development mode:

`npm start`

Note: In Windows, use `cmd.exe` to ensure `--window-enable-inspector` is passed along correctly. 

## Release

`npm run build` to create binaries.
